📱 App Overview

MINUMNOW is a water intake tracking Android app built with Jetpack Compose. It features local storage, cloud sync, live weather data, and GPS integration to provide personalized hydration advice.

💭 Reflection & Learning Journey

📈 Growth from Lab 1 to Project 2

My journey began with Lab 1 where I first encountered Jetpack Compose and struggled with understanding the declarative UI paradigm. By Project 2, I was confidently building 7-screen applications with MVVM architecture, integrating multiple data sources, and working with hardware sensors. The progression from simple UI components to a fully connected, hardware-aware application has been remarkable.

⚡ Challenges Faced

The most challenging aspect was integrating Firebase Firestore while managing asynchronous callbacks alongside Jetpack Compose's reactive UI. Understanding how to bridge Firestore's callback-based API with Compose's state system required deep understanding of coroutines and side effects. Setting up proper Gradle dependencies and resolving version conflicts between Room, Firebase, and Retrofit was also a significant challenge that taught me about Android's dependency management system.

🌍 SDG 3 Impact

MINUMNOW directly supports SDG 3 — Good Health and Well-being by promoting one of the simplest yet most impactful health habits: adequate daily hydration. By providing real-time weather-based hydration advice, the app personalizes recommendations based on environmental conditions. The community tips feature encourages knowledge sharing about healthy hydration practices, creating a collective health-conscious community.
